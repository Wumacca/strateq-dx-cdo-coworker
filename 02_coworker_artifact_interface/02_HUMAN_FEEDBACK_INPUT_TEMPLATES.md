# Human Feedback Input Templates

## Purpose

These templates define what the Digital Lead should feed back after speaking with departments, sponsors, digital champions, vendors, developers, or DRB.

## Template A: Department Clarification

```text
Initiative / Hopper item:
Who I spoke to:
Date:
Department / business area:
Confirmed problem:
What happens today:
What should be different:
System affected:
Process affected:
Owner / sponsor:
Digital champion:
Urgency:
Business exposure:
Evidence provided:
Next action:
```

## Template B: Sponsor / Owner Confirmation

```text
Initiative:
Sponsor / owner:
Confirmed? Yes / No / TBC
Decision authority:
Key comments:
Concerns:
Conditions:
Next action:
```

## Template C: Vendor / Developer Clarification

```text
Initiative:
Vendor / developer:
Question asked:
Answer received:
Feasibility:
Estimated effort / cost if known:
Dependency:
Risk:
Scope implication:
Next action:
```

## Template D: Post-DRB Feedback

```text
Initiative:
DRB date:
Decision:
Conditions:
Amendments required:
Route agreed:
Owner confirmed:
Sponsor confirmed:
Digital champion confirmed:
Cost approval required:
Nitro required:
Next Jira status:
Comments:
```

## Usage Rule

Feed back facts only. Do not rewrite the initiative based on assumptions. If something is still unclear, mark it as TBC.
