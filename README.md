# Strateq DX

Strateq DX is the controlled evidence and delivery repository for Strateq-led digital transformation, governance, capitalisation, and Board reporting work. It is separate from Quantuum AI product build material.

## Repository Structure

### `THREE60/`
Holds THREE60 delivery and governance evidence, organised by capitalisation period. Source markdown files in this folder are the controlled baseline for Board-facing reporting and closeout artefacts.

### `_archive/source_zips/`
Holds original source packs and imported zip files. Board-facing content must be generated from controlled markdown source files, not from standalone slide edits or unarchived imports.
