# THREE60 Operations Digital Capitalisation Programme — Board Closeout Pack

This folder contains the governed markdown baseline for the January to July 2026 Operations Digital Capitalisation Programme closeout and the next six-month capitalisation request.

The files are intended to be committed to GitHub and used as the source of truth for:
- Board closeout reporting
- next capitalisation request preparation
- benefits evidence control
- digital maturity movement evidence
- governance process evidence
- future digital transformation BAU updates

The presentation should be generated from these files. The slide deck must not become the source of truth.

## File order

| File | Purpose |
|---|---|
| `00_SOURCE_INDEX.md` | Source document and evidence map |
| `01_ORIGINAL_CAPITALISATION_REQUEST.md` | What the approved programme set out to do |
| `02_DELIVERABLES_BASELINE.md` | Baseline commitments and roadmap deliverables |
| `03_ACTUAL_DELIVERY_AND_VARIANCE.md` | Delivery performance against baseline |
| `04_PROGRAMME_BENEFITS_AND_EVIDENCE_REGISTER.md` | Programme benefit claims, evidence, and Board-safe wording |
| `05_DIGITAL_GOVERNANCE_PROCESS.md` | Intake-to-hopper-to-delivery-to-adoption governance model |
| `06_EMERGENT_WORK_REGISTER.md` | Additional absorbed work and control position |
| `07_BOARD_CLOSEOUT_REPORT_DRAFT.md` | Draft Board narrative for closeout pack |
| `08_NEXT_6_MONTH_CAPITALISATION_PLACEHOLDER.md` | Placeholder structure for the next request |
| `09_DIGITAL_MATURITY_REGISTER.md` | Baseline-to-current maturity movement register |

## Control principle

Evidence first. Report second. Slides last.
