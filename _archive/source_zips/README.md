# Source Zips Archive

This folder holds original imported source packs and zip files for reference and audit purposes.

Files are archived here when a source pack has been extracted and its contents placed into a controlled folder structure elsewhere in the repository. The zip is retained as an immutable record of what was originally received.

Do not use files in this folder as working source material. All controlled evidence is maintained in the relevant programme folder (e.g. `THREE60/CAPITALISATION_2026_H1/`).
